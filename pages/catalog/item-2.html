Описание товара

Телевизор LED Erisson 19LES76T2 в строгом черном корпусе представляет собой небольшой прибор для просмотра телевизионных программ. Диагональ его экрана равна 19″ (48 см), что делает его удобным устройством для установки в небольших помещениях. При помощи подставки его можно установить на тумбе, а отверстия, поддерживающие стандарт крепления VESA 75×50, позволяют закрепить телевизор на стене. Преимуществом аппарата являются встроенные тюнеры DVB-T, DVB-T2, DVB-C, которые исключают необходимость использования дополнительного оборудования. Для подключения накопителей, внешних дисков или мобильных устройств Erisson 19LES76T2 использует два порта USB. А при помощи входа HDMI можно подключать видео технику. Вывод изображения на экран осуществляется в разрешении 1366×768 (HD), поэтому картинка видео будет четкой, динамичной и насыщенной. Устройство предусматривает выход для наушников, а для управления телевизором используется пульт ДУ.
